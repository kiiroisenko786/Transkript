Time Input Format

  • Enter times in HH:MM:SS:FR format
  • First 3 fields: 00-99
  • Last field (frames): 00-23
  • Start time must be before end time

How to Use

  1. Enter start and end times
  2. Type your transcript text
  3. Click "Add to Results" to save the entry
  4. Click on any result item to edit it
  5. Use "Export to TXT" to save all results to a file

Important Notes

  • Duplicate records are not allowed
  • Results are automatically sorted by start time
  • Exported files use subtitle format
